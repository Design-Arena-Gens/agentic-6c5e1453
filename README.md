# Anon Chat

Anonymous global chat for everyone. Built with Next.js Edge WebSockets and deployable to Vercel.

## Local dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Deploy

Use Vercel CLI (already included as a devDependency):

```bash
npx vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-6c5e1453
```
